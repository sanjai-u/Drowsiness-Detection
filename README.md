# AI Drowsiness Detection System

Real-time drowsiness detection using OpenCV and MediaPipe.

##  Features

* Eye tracking
* Drowsiness alert
* Real-time detection

##  Tech Stack

* Python
* OpenCV
* MediaPipe

##  Setup

```bash
git clone https://github.com/your-username/AI-Drowsiness-Detection.git
cd AI-Drowsiness-Detection
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/drowsy.py
```

##  Note

Supports only for MAC OS
Press `q` to exit.
